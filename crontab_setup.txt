# 每日定时抓取设置说明 (Crontab Setup Instructions)

系统可以使用 Linux 系统的 `cron` 守护进程在每天早晨 6:30 自动执行抓取任务。

## 配置步骤：

1. 打开终端，运行以下命令来编辑当前用户的 crontab 配置文件：
   ```bash
   crontab -e
   ```

2. 如果是第一次编辑，系统会提示您选择编辑器（推荐选择 `nano`，比较简单）。

3. 滚动到文件末尾，添加以下这行配置：
   ```text
   30 6 * * * /home/allenxu/news/run_fetcher.sh
   ```

4. 保存并退出编辑器：
   * 如果使用 `nano`：按 `Ctrl + O` 保存，按 `Enter` 确认，然后按 `Ctrl + X` 退出。
   * 如果使用 `vim`：按 `Esc` 键，输入 `:wq` 并回车。

5. 验证是否配置成功：
   运行以下命令，确认新加的内容已保存在列表内：
   ```bash
   crontab -l
   ```

配置成功后，每天早晨 6:30，系统将会在后台静默抓取新闻，并把结果存入 `digest.db` 数据库。
所有运行日志都会写入同目录下的 `fetcher.log` 中，方便您随时查看抓取状态。
